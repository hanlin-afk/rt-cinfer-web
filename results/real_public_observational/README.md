# Real public observational results

Run `scripts/fetch_pagespeed_crux.py` and then `scripts/run_real_public_observational.py` to generate a real public Web Vitals report from CrUX/PageSpeed field data.

This directory intentionally does not include invented numeric results. Public observations can support external-validity screening and bottleneck analysis. Causal deployment claims require a real intervention log or randomization design.
